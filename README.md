# TAO - Bittensor Miner

Bittensor subnet mining infrastructure.

## Target Subnets
- SN64 Chutes (serverless GPU inference)
- SN12 ComputeHorde (GPU compute rental)
- SN27 Compute (decentralized compute)
- SN8 Taoshi (financial prediction)

## Setup
```bash
pip install bittensor
btcli wallet create
btcli subnet register --netuid <SUBNET_ID>
```
